# 🤖 Baymax 1.0 — Personal Healthcare Companion AI

> *"Hello. I am Baymax, your personal healthcare companion."* 🤍

---

##  What is Baymax?

Baymax is a personal AI health assistant inspired by the lovable robot from Big Hero 6.
He listens, understands, and gently guides you through your health concerns —
in simple, calm, and friendly English.

No drama. No panic. Just care. 🤍

---

##  Features

- 💬 Chat with Baymax about your health concerns
- 🌡️ Describes symptoms and gives gentle guidance  
- 😊 Always calm, friendly and patient — no matter what
- 👊 Fist bump when you feel better — *Balalalala!*
- 🚫 Stays focused on health only — no distractions
- 🤍 Always ends with a caring message

---

##  Built With

- Python
- Google Gemini API
- python-dotenv

---

##  How to Run

```bash
# Activate virtual environment
venv\Scripts\activate

# Run Baymax
python baymax.py
```
